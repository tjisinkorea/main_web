Work experience

I spent the last four years working for Z. It was a small company that offered Audio Visual solutions at the commercial level. The CEO had downsized the company a few years back to focus on another project, and hiring me was an attempt to revitalize Z. When I joined as the Director of Sales, they had a few loyal accounts, but it was struggling.

In my first month, I landed an account with a company that built booths for tradeshows. We were to offer a turnkey solution of renting out our video wall displays and providing installation as well as tech support. This really opened up the door to a new rental market, and this account led to many other accounts in the same booth building sector.

I remember my second account very clearly, as it was a subsidiary of JP Morgan in the real estate market. They wanted an interactive WayFinding kiosk in each of their 6 lobbies that provided map guidance to users, as well as provide other information they wanted to convey. We provided six 40" touchscreen displays connected to Intel NUCs. I created an Amazon EC2 instance of a Windows server so that all 6 computers can be updated remotely. The hard part was creating the Wayfinding software. The next two months I spent with our engineer and designer to create maps of the buildings and creating routes, creating database of tenants, and so much more. But it was a huge sale, and even to this day an important account as they still pay monthly for server usage and support.

Up to this point our strategy was direct sales to businesses and consumers, but I decided to try to work our strategy into channel sales. I contracted manufacturer's reps from different territories and was able to snag with them their reseller/integrator contacts as well. This was a huge change because now integrators called us with project leads.

I had pretty much become the spokesman of the company, dealing with all external requests, making contracts with distributors/integrators/reps, reaching out to vendors and managing key government accounts. I was also responsible for creating bids for government RFQs.

By the time I left Z, I was personally averaging $250,000 per quarter in sales, not too shabby for starting out by cold-calling with zero leads. I'm sure if COVID hadn't struck, and I hadn't left sales would have kept climbing YoY, especially because our "channel" network was becoming well established and I had just completed Z's GSA contract which listed us onto the US government procurement databases as a trusted manufacturer.

Because of these experiences at Z, my knowledge in company operations is very rounded. I believe that this gives me an edge and also the confidence to be able to take on a variety of roles, whatever they may be, at my next opportunity. I accept and await my newest challenge!

What I'm learning.

August 1, 2021

I did something fun over the weekend. I ordered a RaspberryPi4 8GB RAM from Amazon... That's right, to Korea. It shipped in about 10 days, and there was no import fees, and only $8 for shipping. It was a better deal than buying one locally.

I messed around with some different builds. I tried the RETROPI emulator to play oldschool games. Neat, but I don't want to invest any time playing those oldskool video games. I tried a few different flavors of Linux operating systems. I tried a build where I installed VMWare esxi hyperviser so I can run multiple virtual machines simultaneously from my single Pi4. I thought this was going to be my permanent build, and it probably will, but I got a little impatient with Linux Servers taking so long to set up via USB storage.

So I decided to setup Raspbian OS and install Pi-Hole. Pi-Hole allows me to set up a custom local DNS that blocks ALL advertisements. I SSH into my pi4 to access it's command line. Then here are the steps:

  • Install Pi-Hole - sudo curl -sSL https://install.pi-hole.net | bash
  • Install Unbound DNS - sudo apt install unbound
  • Create Unbound Configuration File - sudo nano /etc/unbound/unbound.conf.d/pi-hole.conf
  • Copy example config - https://docs.pi-hole.net/guides/dns/unbound/
  • Restart Unbound to apply Configuration - sudo service unbound restart
  • Disable Forwarding DNS in PiHole
  • Set Custom DNS in PiHole - 127.0.0.1#5335
  • Set custom DNS to my pi4 in any device that I want to block ads

Not only does it block ads, I get a log of all the queries. Also, I can manually set whitelists and blacklists of domains. If I wanted to block certain sites like netflix or youtube to prevent people on my network from joining, I could. But If I wanted to authorize this totalitarian internet regime, I would have to change the DNS on my router, not on individual devices.

It's pretty neat. No more ads on web browsers or Apps.
